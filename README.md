# Organizador de arquivos

Organizador de arquivos, programado em CSharp
Procura por arquivos em um diretório e move eles para pastas organizadas por extensão.

## Instruções

1. Verifique se você tem o .NET instalado, usando o comando:

``````cmd
dotnet --version
```

Se o comando não retornar um número de versão, baixe e instale o [.NET](dotnet.microsoft.com).

2. Extraia o conteúdo do repositório para um diretório de sua preferência.

3. Abra o arquivo sort.cs, e altere a string 'target' para um diretório de sua preferência.

4. Abra um terminal na pasta onde sort.cs está localizado, e execute o comando:

```cmd
dotnet run
```
